The model: vocabulary types make states explicit

TypeMeaningPrimary risk
string_view/spanBorrowed, bounded observationOwner dies or storage invalidates

RAII valueOwned state with scoped lifetimeAccidental expensive copies if API is vague

NetForge's parser borrows the request while tokenizing, validates grammar and key policy, then returns an owned Command variant or ParseError. This prevents a worker from retaining views into a connection buffer that has already died. Review the standard draft sections for span and variant, plus the lifetime reference.
