# Your NetForge work tree

Edit this directory. It starts as a compiling but deliberately incomplete
implementation; the day-labelled tests stay red until you earn each behavior.
The public interfaces are fixed so that all tests can compile from Day 1.

Run one gate from the repository root:

```bash
./scripts/check.sh work day01
```

Search for `TODO(dayNN)` to find the intended surface for a lesson. Resist the
urge to implement future days early: later tests encode failure behavior you
have not modeled yet.
