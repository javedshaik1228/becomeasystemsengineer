Mental model

GTC remainders may become makers. IOC remainders are reported but never rest. A live-ID index and the book must agree: an ID is either present in both or absent from both.

Build

  1. Rest only GTC remainders.
  2. Reject duplicate live order IDs.
  3. Track live IDs and implement cancel.
  4. Remove empty levels and index entries after fills and cancellation.
  5. Document baseline cancel complexity and one faster alternative.

Break it deliberately

Cancel an unknown ID, cancel the remainder after a partial fill, submit a duplicate, and send an oversized IOC against one small maker. Ensure no IOC bid appears afterward.
