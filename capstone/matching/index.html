What you will ship

01 · Matching core

Price-time priority

FIFO queues at each price level, maker-price execution, GTC/IOC behavior, cancellation, and explicit acknowledgements.

02 · Concurrency

Lock-free ingress

A bounded sequence-number ring buffers orders with back-pressure while one consumer keeps book mutation deterministic.

03 · Edges

Redis + Kafka

Publish fills through RESP2 Redis and optional librdkafka, without putting broker stalls on the matching path.

04 · Evidence

Replay and benchmark

Generate deterministic CSV orders, replay them, run unit/ingress/mock tests, and measure throughput.

Independent by design. This project does not depend on completing the 14-day NetForge course. Use it as a portfolio artifact, interview deep dive, or separate systems-building track.
